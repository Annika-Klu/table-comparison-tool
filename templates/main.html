Compare Tables

This tool compares two versions of a table based on each entry (row) having a unique key value.
It identifies the differences between the entries and lists those found in one of the tables but not the other.

Requirements

    The comparison relies on the following criteria:
  • each entry must have a key value that is unique to it. This may be a number (e. g. an order number), but it may as well be a word, or a combination of both
  • the key value must be in the first column of each table
  • the columns you wish to compare must have the same names in each table

Upload files

Upload files in .csv, a simple data matrix format that every spreadsheet program should be able to save to.
In this format, values are separated by a predefined character called the delimiter. This tool supports the following delimiters: : ; ,
If possible, save the CSV files using a common encoding standard such as UTF-8 that can handle characters of many different languages.
If your tables are in different sheets of the same document, save each sheet to a single CSV file.
