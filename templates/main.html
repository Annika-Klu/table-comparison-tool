Compare Tables

This tool compares two tables based on each entry having a unique key value.
It identifies the differences between the entries and lists those found in one of the tables but not the other.

Requirements

    The comparison will work if the following criteria apply:
  • each entry must have a key value that is unique to it. This may be a number (e. g. an order number), but it may as well be a word, or a combination of both
  • the key value must be in the first column of each table
  • the columns you wish to compare must have the same names in each table
